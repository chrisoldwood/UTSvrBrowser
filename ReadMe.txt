UT Server Browser v1.0 Alpha
============================

Introduction
------------

This is a browser for querying servers running the original version of Unreal
Tournament. It knows about certain popular mods and provides a number of ways
to filter the server list. You can also manage your UT server favourites via
the tool too.

Releases
--------

Stable releases are formally packaged and made available from my Win32 tools page:
http://www.chrisoldwood.com/win32.htm

The latest code is available from my GitHub repo:
https://github.com/chrisoldwood/UTSvrBrowser

Installation
------------

Copy the files UTSvrBrowser.exe, UTSBMods.ini and UTSvrBrowser.ini to a folder.

Uninstallation
--------------

Manually delete the files copied above.
